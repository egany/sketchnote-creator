# Quality Gate

Score internally from 1 to 10.

## 1. Message Clarity

The viewer should understand the main point in 3 seconds.

Reject if the visual has more than one competing message.

## 2. Visual Impact

The image needs a strong metaphor or structure, not just decorative bullets.

Reject if it looks like a plain slide with icons.

## 3. Readability

Text must be short, large enough, and easy to scan on mobile.

Reject if labels are too long, dense, or likely to be distorted by image generation.

## 4. Style Consistency

The prompt must match the selected template style.

Reject if it mixes incompatible styles such as photorealistic + notebook sketchnote + watercolor unless the selected style requires it.

## Pass Rule

Pass only when total score is at least 32/40 and no item is below 8.

If the score fails, simplify layout, reduce text, strengthen metaphor, or pick a better style.

# Style Guide

Sketchnote Creator uses eight style families. Each style exists to make a different kind of content easy to understand quickly.

Use fewer words than a normal slide. Prefer visual hierarchy, arrows, icons, metaphors, and short labels.

Use EIS for educational infographic posters: cheat sheets, concept explainers, terminology guides, and framework overviews with 6-9 short concept cards.

For Vietnamese visuals, use natural, accented Vietnamese and avoid long in-image sentences.

# Installation

## Codex Desktop / Agent Skills

Copy the installable skill folder into your global skills directory.

Windows PowerShell:

```powershell
Copy-Item -Recurse .\skill\sketchnote-creator $HOME\.agents\skills\sketchnote-creator
```

Then restart Codex.

## Manual Install

The installable skill is:

```text
skill/sketchnote-creator
```

Copy that folder into the skill directory used by your agent runtime.
